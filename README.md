What is computer network:
-Its a strucutre in which atleast 2 devices communicate with each other

Purpose of computer network:
-Image and sound transfer
-Hardware and software sharing
-file and data sharing
-central management

The Advanced Research Projects Agency Network (ARPANET) is a computer network considered the foundation of the internet.

Types of Networks:
-PAN(personal area network)
-LAN(Local area network)
-MAN(Metropolitan area network)
-WAN(Wide area network)
-CAN(Campus area network)
